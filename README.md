Gerando notebook
----------------------

Requerimentos: [Python 3](https://www.python.org/), [pdflatex](http://pdftex.org)

```bash
sudo apt install python3
```

```bash
sudo apt install texlive texinfo texlive-fonts-recommended texlive-latex-extra
```

Comando:

```bash
python3 notebookcreator.py
```
